# 🔐 VeilText Encoder

A lightweight text encoding tool built using HTML, CSS, and JavaScript. VeilText Encoder transforms plain text into an obfuscated encoded format using custom character manipulation and base36 conversion, giving a secure and hacker-style feel.

---

## 🚀 Features

- 🔐 Custom text encoding algorithm  
- 🔢 Character transformation using ASCII logic  
- 🔄 Base36 encoding for compact output  
- ⚡ Instant conversion  
- 🎨 Hacker-style UI design  
- 💻 Lightweight and fast (no external libraries)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)

1. Enter your text in the input box  
2. Click "Convert"  
3. Encoded output will appear instantly  
4. Copy and use encoded text  

---

## 🔐 How It Works

- Each character is converted to ASCII code  
- A transformation is applied:  
  (code + 777) × 3  
- Result is encoded using base36  
- Final output is joined using "."
